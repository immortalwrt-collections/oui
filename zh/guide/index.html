# 介绍

oui使用vue.js和element-ui实现，灵感来自于LuCI2。

oui使用json-rpc和OpenWrt子系统通信。通过json-rpc调用ubus。通过ubus存取各种系统数据(通过uhttpd-mod-ubus提供基于HTTP的接口API)。

# 特性

  • 快: 使用静态页面；通过ubus存取各种系统数据
  • 多用户: 支持多用户及权限管理
  • Vue + Element-UI 加持: 享受 Vue + Element-UI 的开发体验
